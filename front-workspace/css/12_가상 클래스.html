# Pseudo Class (가상 클래스)

  • 선택자 뒤에 추가로 사용하여 해당 요소의 특정 상황을 선택하는 것
  • 선택자 뒤에 :을 붙여 사용한다

# 가상 클래스의 종류

  • :first-child
  • :nth-child(n/even/odd)
  • :last-child
  • :first-of-type
  • :nth-of-type(n/even/odd)
  • :last-of-type

# 링크에 적용되는 가상 클래스 사용해보기

  • :link - 해당 요소가 방문한 적이 없는 링크 상태일 때
  • :visited - 방문한 적이 있는 상태일 때
  • :active - 해당 링크를 누르고 있는 상태일 때
  • :hover - 해당 요소에 마우스를 올린 상태일 때

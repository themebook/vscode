# display: grid

  • column과 row를 설정하여 사용할 수 있는 display 속성
  • display 속성을 grid로 설정한 요소는 grid-box가 된다
  • 내부의 자식 요소들은 grid-item이 된다

# display: grid에서 사용하는 속성들

  • grid-template-columns: 열의 개수와 각 열의 크기를 정의하는 속성
  • grid-template-rows: 각 행의 크기를 정의하는 속성
  • column-gap: 각 열의 간격을 설정한다
  • row-gap: 각 행의 간격을 설정한다

# grid-item이 사용하는 속성들

  • grid-column-start: 해당 칸이 몇 번째 열에서 시작하는지 설정할 수 있다
  • grid-column-end: 해당 칸이 몇 번째 열에서 끝나는지 설정할 수 있다
  • grid-row-start: 해당 칸이 몇 번째 행에서 시작하는지 설정할 수 있다
  • grid-row-end: 해당 칸이 몇 번째 행에서 끝나는지 설정할 수 있다
  • order: 해당 칸의 배치 순서를 설정할 수 있다

# 하나의 속성에 여러개의 값을 부여하는 경우

  • 한 속성에 값을 하나만 주는 경우 (모든 방향에 적용됨)
  • 한 속성에 값을 두 개 주는 경우 (상하/좌우)
  • 한 속성에 값을 세 개 주는 경우 (상/좌우/하)
  • 한 속성에 값을 네 개 주는 경우 (상/우/하/좌)

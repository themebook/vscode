# 웹 페이지에 CSS를 적용하는 3가지 방법

1. inline CSS (해당 요소의 스타일 attribute 사용하기)
- 해당 요소의 style 속성에 스타일을 직접 정의하는 방식
- CSS 적용 우선 순위가 가장 높다
2. internal CSS (파일 내부의 CSS)
- 같은 파일의 head에 style 태그로 스타일시트를 직접 정의하는 방식
- 더 나중에 작성한 스타일시트가 우선 순위를 가지게 된다
3. external CSS (외부의 CSS를 끌어다 사용)
- 다른 CSS파일을 현재 웹 페이지로 불러서 사용하는 방식
- link 태그를 사용해 외부의 CSS파일을 현재 페이지에 연결한다
- 우선 순위는 internal CSS와 같다

# display property

  • 웹 브라우저가 해당 요소를 어떤 방식으로 그려야하는지 결정하는 속성
  • 대부분의 HTML 태그들은 기본적으로 display 속성이 block 또는 inline으로 설정되어있다

# block 요소

  • 너비가 남더라도 항상 한 줄을 모두 차지한다
  • 기본적으로 설정된 width는 100%이다
  • padding과 margin을 적용할 수 있다
  • 박스 취급을 당하는 요소이다
  • ex> div, p, h,ul, ol, dl, dd, dt, li... 등의 태그들은 기본 display 속성이 block이다

# inline요소

  • 영역을 지정하더라도 글의 내용을 해치지 않는다
  • 글 내용에서 특정 부분을 선택하기 위한 용도로 많이 사용한다
  • padding과 margin이 올바르게 적용되지 않는다
  • 박스 취급이 아닌 글자 취급을 당한다
  • ex> span, b, a, i... 등은 기본 display속성이 inline이다
# span의 기본 속성은 inline이지만 바꿔서 사용할 수 있다

# display: none
  • 해당 요소를 그리지 않는다
  • 해당 요소가 차지하고 있던 영역도 사라진다

# visibility: hidden

  • 해당 요소를 안보이게 설정한다
  • 해당 요소가 차지하고 있던 영역이 그대로 유지된다
  • 단순하게 보이고 안보이고만 결정하는 속성이기 때문에 다양한 옵션이 존재하지 않는다

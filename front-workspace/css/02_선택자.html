# CSS 선택자 (Selecter)

  • 태그이름: 해당 태그 이름을 가진 모든 태그를 선택
  • #아이디: 해당 id속성 값을 지닌 요소를 선택 (하나만 선택)
  • .클래스: 해당 class 속성 값을 지닌 모든 요소를 선택 (여러개 선택)
  • * : 모든 요소 선택
  • , : 여러개의 선택자를 사용
  • > : 왼쪽 요소의 자식 요소인 오른쪽 요소들을 선택
  • 공백 : 왼쪽 요소의 자손 요소인 오른쪽 요소들을 선택

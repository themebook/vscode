# 절대 길이 단위

  • 환경(웹 브라우저의 크기 등)이 변하더라도 항상 일정한 크기를 가지는 단위
  • px: 픽셀(화소)
  • mm: 밀리미터
  • cm: 센티미터
  • in: 인치
100px
50mm
5cm
5in

# 상대 길이 단위

  • 환경이 변하면 크기가 함께 변하는 단위들
  • %: 부모 요소의 몇 %를 차지하는지 설정 (부모 요소 크기에 따라 설정됨)
  • em: 현재 요소의 font-size 값에 비례한 크기 설정
  • rem: html 태그의 font-size 값에 비례한 크기 설정
  • vw: viewport width, 현재 화면 너비에 비례한 크기 설정 (1vw는 현재 화면 너비의 1%크기를 의미함)
  • vh: viewport height, 현재 화면 높이에 비례한 크기 설정 (1vh는 현재 화면 높이의 1%크기를 의미함)
50%
10em
10rem
20vw
20vh

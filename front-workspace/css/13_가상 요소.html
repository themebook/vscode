# Pseudo Element (가상 요소)

  • 해당 요소의 특정 부분에 있는 가상의 요소를 선택하는 선택자
  • 선택자 뒤에 ::을 붙여 사용한다
  • ::first-letter - 첫 번째 글자 부분을 선택
  • ::first-line - 첫 번째 줄을 선택
  • ::selection - 블록 지정한 부분을 선택
  • ::before - 해당 요소 시작하는 곳에 있는 가상의 요소를 선택
  • ::after - 해당 요소가 끝나는 곳에 있는 가상의 요소를 선택

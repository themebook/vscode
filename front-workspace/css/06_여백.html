# HTML Box Model

  • HTML은 모든 요소를 박스 형태로 구성한다
  • 제일 안쪽에 content-box가 있고, content-box를 둘러싼 padding, border, margin으로 요소의 크기와 위치를 결정한다
  • 기본적으로(box-sizing: content-box일 때) 요소 전체의 크기는 content-box의 크기와 padding과 border의 굵기를 더한 값이 된다
  • box-sizing 속성을 border-box로 설정한 후에는 width와 height값 만으로 요소 전체의 크기(border-box의 크기)가 결정된다

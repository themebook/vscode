# input

  • input 태그를 사용해 사용자로부터 다양한 입력을 받을 수 있다
  • input 태그의 type 속성으로 어떻게 입력받을지 결정할 수 있다
  • input 태그의 value 속성에 값을 설정해 놓으면 기본값이 설정된다
  • input 태그의 placeholder 속성의 값은 무엇을 입력해야하는지 알려준다
  • input 태그에 readonly 속성을 추가해놓으면 읽기 전용이 된다
  • input 태그에 disabled 속성을 추가해놓으면 비활성화 된다
타입 예시 설명
type="text" 일반적인 텍스트를 입력하는 곳
type="number" 숫자를 입력할 수 있는 곳
type="password" 비밀번호를 입력할 수 있는 곳 (비밀번호를 가려준다)
type="checkbox" 관심 분야





같은 카테고리 내에서 여러가지를 선택할 수 있는 버튼
type="radio" 맛을 선택하세요!



같은 카테고리 내에서 하나만 선택할 수 있는 버튼
type="color" 색상을 선택할 수 있다
type="date" 날짜를 선택할 수 있다
type="file" 사용자의 컴퓨터에서 원하는 파일을 선택한다
type="hidden" 사용자에게는 보여지지 않는 상태로 데이터를 보관하기 위해 사용한다
<textarea> 많은 양의 텍스트를 입력하기 위해 사용하는 태그
<select>, <option> 주어진 항목들 중 하나를 선택할 수 있는 태그

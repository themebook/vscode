- HTML은 HyperText Markup Language의 약자입니다
- HTML은 웹 페이지의 생김새를 묘사하는 마크업 언어입니다
- HTML을 해석하여 우리에게 그려주는 프로그램을 웹 브라우저라고 한다
- 웹 브라우저가 html의 내용을 보고 해석하여 화면에 그려주는 작업을 렌더링이라고 합니다
- 웹 브라우저의 종류로는 Chrome, Safari, Whale, Edge, Firefox, Internet Explorer등이 있습니다

# 01_Javascript

  • 웹 점유율 100%를 자랑하는 프로그래밍 언어
  • 웹 브라우저가 해석하여 동작하는 프로그래밍 언어 (웹 브라우저에는 자바스크립트 해석기가 포함되어 있다)
  • html은 웹 페이지의 뼈대를 담당하고 css는 웹 페이지의 외형을 담당하고 javascript는 웹 페이지의 동작을 담당한다

# javascript 적용 방식 3가지

  • inline: HTML의 이벤트 속성(on으로 시작하는 속성들)에 직접 추가하는 방식
  • internal: 현재 페이지의 script 태그에 적용하는 방식
  • external: 다른 자바스크립트 파일을 불러와서 사용하기
